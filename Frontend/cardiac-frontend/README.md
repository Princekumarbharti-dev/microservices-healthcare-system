# Cardiac Diagnosis System - Frontend

This is the frontend application for the **Cardiac Diagnosis System** built using **React, TypeScript, and Vite**.  
It provides a user interface to interact with backend microservices such as authentication, diagnosis data, bookmarks, and user profiles.

---

## Tech Stack

- React
- TypeScript
- Vite
- Material UI
- Axios

---

## Features

- User Login & Authentication
- View Cardiac Diagnosis Records
- Filter diagnosis data
- Bookmark diagnosis results
- View previous bookmarks
- User profile management
