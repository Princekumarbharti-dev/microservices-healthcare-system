# Cardiac Diagnosis System

A microservices-based healthcare application for managing and analyzing cardiac diagnosis data. The system integrates with an external Diagnosis API and provides secure JWT authentication, diagnosis search and filtering, user profile management, and bookmarking of important diagnosis records.

## Features

- JWT-based Authentication
- Search and Filter Diagnosis Data
- Bookmark Management
- User Profile Management
- Password Reset
- Guest User Access
- External Diagnosis API Integration
- Microservices Architecture

## Tech Stack

- React
- Spring Boot
- Spring Security (JWT)
- Eureka Service Discovery
- API Gateway
- Kafka
- Docker
- MySQL
